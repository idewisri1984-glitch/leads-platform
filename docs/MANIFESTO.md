# Bali Leads Platform Manifesto

Software should help people create relationships.

Not spreadsheets.

Not databases.

Not reports.

Relationships.

Every company in the system represents a real business.

Every contact represents a real person.

Every workflow represents an opportunity.

Artificial Intelligence should never replace human relationships.

It should remove repetitive work so people can focus on communication, trust and value.

The platform exists for one reason:

Help businesses find businesses.

Everything else is implementation details.
